# customer_behaviour
Customer Shopping Behavior Analysis using Python, SQL, and Data Visualization. This project includes data cleaning, EDA, and business insights.
Customer Behavior Analysis Project
🔹 Overview

This project focuses on analyzing customer shopping behavior to generate meaningful business insights.
It covers the complete data analytics workflow — from data loading and cleaning to visualization and reporting.

The goal is to understand customer patterns such as purchasing habits, spending behavior, and segment-wise performance.

🔹 Dataset
Dataset contains customer-level shopping data
Includes features like:
Customer ID
Age, Gender
Product Category
Purchase Amount
Subscription Status
Frequency of Purchases
Discounts & Promo Usage
🔹 Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)
SQL (PostgreSQL / SQL Server)
Power BI (Dashboard & Visualization)
Jupyter Notebook (Analysis)
Gamma (Presentation creation)
🔹 Project Steps
1. Data Loading
Imported dataset using Python (Pandas)
Initial exploration using .head(), .info(), .describe()
2. Data Cleaning
Handled missing values
Fixed data types
Removed duplicates
Created new features (e.g., age groups, purchase frequency in days)
3. Exploratory Data Analysis (EDA)
Analyzed:
Customer demographics
Purchase trends
Category-wise sales
Impact of discounts & subscriptions
Used visualizations for better understanding
4. SQL Analysis
Loaded cleaned data into PostgreSQL / SQL Server
Wrote queries to:
Calculate total revenue
Compare customer segments
Analyze purchase frequency
Rank top categories/items
5. Dashboard (Power BI)
Built interactive dashboard including:
Revenue overview
Customer segmentation
Sales by category
Purchase trends
6. Reporting & Presentation
Created a structured report
Built a presentation using Gamma
Highlighted key insights and business recommendations
🔹 Dashboard Highlights
📈 Total Revenue & Average Spend
👥 Customer Segmentation (New / Returning / Loyal)
🛒 Category-wise Performance
🎯 Impact of Discounts & Promotions
🔹 Key Results / Insights
Subscription customers tend to spend more
Discounts increase purchase frequency
Certain categories generate higher revenue
Loyal customers contribute significantly to total sales
🔹 How to Run
1. Clone the Repository
git clone https://github.com/your-username/customer_behaviour.git

pip install pandas numpy matplotlib seaborn sqlalchemy psycopg2
3. Run Jupyter Notebook

4. Database Setup
Create database in PostgreSQL / SQL Server
Update connection credentials in the script
Run SQL queries
5. Power BI
Open .pbix file
Refresh data if needed
