# customer_behaviour
Customer Shopping Behavior Analysis using Python, SQL, and Data Visualization. This project includes data cleaning, EDA, and business insights.
# 📊 Customer Behavior Analysis

## 🔹 Overview
This project analyzes customer shopping behavior to generate meaningful business insights.  
It covers the complete data analytics workflow — from data loading and cleaning to visualization and reporting.

---

## 🔹 Dataset
The dataset contains customer-level shopping data with features such as:
- Customer ID
- Age, Gender
- Product Category
- Purchase Amount
- Subscription Status
- Frequency of Purchases
- Discounts & Promo Usage

---

## 🔹 Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (PostgreSQL / SQL Server)
- Power BI
- Jupyter Notebook
- Gamma (Presentation)

---

## 🔹 Project Steps

### 1. Data Loading
- Imported dataset using Pandas
- Used `.head()`, `.info()`, `.describe()`

### 2. Data Cleaning
- Handled missing values
- Fixed data types
- Removed duplicates
- Created new features

### 3. Exploratory Data Analysis (EDA)
- Customer demographics
- Purchase trends
- Category-wise sales
- Impact of discounts

### 4. SQL Analysis
- Loaded data into PostgreSQL / SQL Server
- Wrote queries for revenue, segmentation, ranking

### 5. Dashboard (Power BI)
- Revenue overview
- Customer segmentation
- Sales trends

### 6. Reporting
- Created report
- Built presentation using Gamma

---

## 🔹 Key Insights
- Subscription customers spend more
- Discounts increase purchase frequency
- Loyal customers contribute most revenue
